# NetEx.Hooks

Provides global hooks for capturing keyboard and mouse events.

## API

API documentation is available [here](https://peckmore.github.io/NetEx).

## Compatibility

| Product                   | Versions              |
|---------------------------|-----------------------|
| **.Net**                  | 5.0+ *(Windows only)* |
| **.Net Framework**        | 2.0 to 4.8.1          |
